Propositional Calculus
Modus Ponens A,A \then B \entails B
Simplification \entails A \then B \then A
Frege \entails (A \then B \then G) \then (A \then B) \then (A \then G)
Contraposition \entails (\not A \then \not B) \then A \then B
Equivalence Introduction \entails \not ((A \then B) \then \not (B \then A)) \then (A \iff B)
Equivalence Elimination \entails (A \iff B) \then \not ((A \then B) \then \not (B \then A))
Conjunction \entails A \and B \iff \not(A \then \not B)
Predicate Calculus
Quantifier Introduction \entails A \then \forall a A
Quantifier Elimination \entails (\forall a A[a]) \then A[b]
Quantifier Distribution \entails (\forall a (A \then B[a])) \then A \then (\forall a B[a])
Existence \entails (\exists a A[a]) \iff \not \forall a \not A[a]
Constant Expressions \entails \forall x forall y A[x,y] \iff \alpha[x,y]
Constant Variable \entails \exists x A[x] \then A[\alpha]
Set Variables
Equality Introduction \entails \forall a \forall b (\forall c c \in a \iff c \in b) \then a = b
Equaility Elimination \entails \forall a \forall b a = b \then A[a,a] \then A[a,b]
Existence \entails \exists a a = a
(Modified) ZF
Axiom of Replacement \entails \forall A (\forall a \forall b \forall c A[a,b] \and A[a,c] \then b = c) \then \forall a \exists b \forall c \forall d c \in a \and A[c,d] \then d \in b
Axiom of Sequences \entails \forall A (\forall a \forall b \forall c A[a,b] \and A[a,c] \then b = c) \then \forall a \exists b a \in b \and \forall c \forall d c \in b \and A[c,d] \then d \in b
Axiom of Power Sets \entails \forall a \exists b \forall c (\forall d d \in c \then d \in x) \then c \in b
Axiom of Unions \entails \forall a \exists b \forall c (\exists d c \in d \and d \in a) \then c \in b
